# Relationship Quiz Game (HTML)

This is a two-player relationship quiz that helps partners understand each other better through values + conflict-style questions, private answers, and a reveal step.

## How to open the game
### Option 1 (Recommended): Open the GitHub Pages link
1. Go to the website link for this repo (GitHub Pages).
2. The game will load in your browser.

### Option 2: Open the HTML file directly
1. Download the repo (or the HTML file).
2. Double-click the HTML file to open it in a browser (Chrome recommended).

## How to play
1. Enter Player 1 and Player 2 names (optional).
2. Click **Start Game**.
3. Each player selects an answer (answers stay hidden).
4. Click **Reveal Answers** to see both answers.
5. Talk through the prompt.
6. Click **Next Question** to continue.
7. Use the **Chat / Reflection Log** to write notes.
8. Click **Export Session** to download your session.

## What the buttons do
- **Start Game**: begins the quiz
- **Reveal Answers**: shows both playersâ€™ answers after both have answered
- **Next Question**: moves to the next question (only after reveal)
- **Export Session**: downloads your session (answers + chat)
- **Clear Chat**: clears the chat only
- **Reset All**: resets everything

## Privacy
This app saves progress in your browser (local storage).  
Nothing is uploaded anywhere unless you add your own backend/API.
